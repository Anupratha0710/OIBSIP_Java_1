🖥️ Online Exam System (Java)

📌 Project Description

The Online Exam System is a Java-based application that allows users to take exams online. It includes features like login authentication, multiple-choice questions (MCQ), and result evaluation.

This project is useful for conducting simple online tests in an efficient and user-friendly way.

---

🚀 Features

-  User Login System (Username & Password)
-  Multiple Choice Questions (MCQ)
-  Timer-based Exam (optional)
-  Automatic Result Calculation
- Simple Console-Based Interface

---

🛠️ Technologies Used

-  Java (JDK 8 or above)
-  MySQL (for database, if used)
-  VS Code (IDE)
-  JDBC (Java Database Connectivity)

---

📂 Project Structure

OnlineExamSystem/
│── Main.java
│── Login.java
│── Question.java
│── Result.java
